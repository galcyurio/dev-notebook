# Vim 단축키 모음

- Home (non-whitespace) : `^`, `_`
- Home (whitespace) : `0`
- End : `$`

- Home + `Insert mode` : `I` (Insert mode at beginning of line)
- End + `Insert mode` : `A` (Append)

- Page Up (half) : `Ctrl + U`
- Page Down (half) : `Ctrl + D`
- Page Up : `Ctrl + B`
- Page Down : `Ctrl + F`

- Pane Up : `Ctrl + Y`
- Pane Down : `Ctrl + E`

- Move to first line : `gg`
- Move to last line : `G`
